EPICS driver for Turbo PMAC2-VME Ultralite board


Before using the Turbo PMAC2-VME Ultralite controller with EPICS, one has to load many parameters through the serial connection with the computer running PEWIN32PRO program under Windows operating system. These settings has to be stored to the controller's non-volatile flash memory with the SAVE command.

1. Settings for Mailbox and DPRAM base addresses.
2. Settings for MACRO link.
3. Settings for step and servo motors.
4. Settings for digital I/O accessory 11E.
5. Settings for interpolator accessory 51E.
6. Settings for analog I/O accessory 59E.
7. Settings for digital I/O accessory 65E.

These settings are in the GMCA_32axis_macro_setup.pmc file.
